# HIT Tracker API

ASP.NET Core Web API built around Mike Mentzer's High Intensity Training philosophy.

## Tech Stack
- ASP.NET Core 10
- SQLite + Entity Framework Core
- JWT Authentication + BCrypt
- xUnit Tests
- Scalar API Documentation

## Getting Started

### Prerequisites
- .NET 10 SDK

### Run
```bash
cd HitTrackerAPI
dotnet run
```
